Specifications / Mining

Titcoin is an independent altcoin based on the Bitcoin Core codebase (v0.16.3).

Ticker:TIT
Hashing algorithm:SHA256d
Target block time:60 seconds
Retargeting algorithm:Modified Digishield
Maximum coin supply:69,000,000
Current coin supply (est.):63,000,000
Genesis block time:Jan 20, 2014
Halving rate:500,000 blocks (347 days)
Current block reward:8.625 TIT
