Download FAQ

Do you have a mobile wallet (Android, iOS)?
No, not yet. Our goal is to be supported by various multi-coin mobile wallets in the future.
Is the wallet safe to use?
Yes. The wallet is completely open source. If in doubt, you can download the source code at Github and compile it yourself!
Do I have to register somewhere?
No, Titcoin can be used completely anonymously! If you want to trade Titcoin for Bitcoin later, you might have to register at a cryptocurrency exchange.
Should I use the setup or the standalone wallet for Windows?
The standalone wallet is optimal for trying out Titcoin. If you want to keep using it, you can install Titcoin to your system using the setup installer.
